# Smart Creative Generator

AI Poster & Thumbnail Studio is a Next.js, React, Tailwind, and Fabric.js web app for creating editable public-awareness campaign creatives, thumbnails, posters, banners, standees, pamphlets, and official social media designs.

## Local Development

```bash
pnpm install
pnpm dev
```

Copy `.env.example` to `.env.local` and configure an AI provider when you are ready to connect real generation services. The app includes a local deterministic generator fallback so the editor remains usable without credentials.
